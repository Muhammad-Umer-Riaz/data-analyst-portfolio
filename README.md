# 📊 Data Analyst Portfolio

A collection of end-to-end data analysis projects covering real-world business problems.
Each project includes data cleaning, exploratory analysis, statistical testing,
and actionable insights — built with Python, SQL, and pandas.

---

## 🗂️ Projects

| # | Project | Description | Tools |
|---|---------|-------------|-------|
| 01 | [Retail Inventory Optimization](./Retail_Inventory_Optimization/) | Analysed ASOS product data to quantify revenue lost due to out-of-stock items ("phantom revenue"). Identified high-impact brands and products using stockout rates, price analysis, and a strategic quadrant visualisation. | Python, pandas, matplotlib, seaborn |
| 02 | [Supplier Performance & Stockout Analysis](./Supplier_Performance_Analysis/) | Analysed 12 months of simulated SAP ERP procurement data across 100 vendors and 500 materials. Quantified $20.99M in delayed procurement spend (62.8% of total), reconstructed historical stockouts from raw movement data, and built a composite supplier scorecard ranking all vendors across four weighted KPIs. | Python, pandas, SQL, sqlite3, matplotlib, seaborn, scipy |
